# TSP-Nearest-Insertion-3-opt
A practical comparison of greedy construction heuristics and local search improvement methods for the Traveling Salesman Problem, benchmarked on TSPLIB datasets.
Algorithms
Construction heuristics

Nearest Insertion (NI)
Cheapest Insertion (CI)
Farthest Insertion (FI)

Local search

2-opt
3-opt

These are combined into 7 pipelines: NI, NI+2opt, NI+3opt, CI, CI+3opt, FI, FI+3opt.
